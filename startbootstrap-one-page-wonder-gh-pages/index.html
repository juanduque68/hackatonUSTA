¿Qué es Entorno?

Entorno es un proyecto que busca llevar la educación a toda la población colombiana y ayudar a empoderar a sus habitantes en temas de tecnología. De esta forma queremos que Colombia sea más productiva.

Aprende con nosotros

Te proveemos las herramientas y la plataforma para que aprendas a programar, desarrollar, crear y aprender mucho sobre tecnología.

Trabaja con nosotros

Lógica de programación


La lógica es la base de todo el conocimiento en programación, ya que es posible aprender a escribir un código para que la computadora interprete correctamente. Es decir que aprendes a comunicarte con la máquina a partir de un lenguaje.


Programación orientada a objetos


La programación Orientada a objetos (POO, u OOP según sus siglas en inglés) es un paradigma de programación que usa objetos en sus interacciones, para diseñar aplicaciones y programas informáticos.


Python


Python es un lenguaje de programación interpretado cuya filosofía hace hincapié en la legibilidad de su codigo. Se trata de un lenguaje de programación multiparadigma, ya que soporta orientación a objetos, programación imperativa y, en menor medida, programación funcional. Es un lenguaje interpretado, dinámico y multiplataforma.

Desarrollo de formulario en HTML5


Requiero un desarrollo de un formulario de evaluación con 17 ítems y 5 niveles de puntuación para cada ítem, de manera que sea realizado un cálculo automático cada vez que se marque o desmarque una puntuación, por lo que las puntuaciones pueden estar con un control tipo checkbox. El formulario debe almacenar nombre, fecha, hora, ciudad, lugar, y evaluador.
El objetivo es que 5 evaluadores puedan almacenar las evaluaciones de manera simultánea.
Se requiere una interfaz para Evaluador y otra para Administrador, donde se puedan visualizar las puntuaciones almacenadas. Las interfaces deben permitir configurar logotipos o imágenes de un evento. Se adjunta una imagen del formulario como una aplicación de Windows.

Habilidades: Angular.js, node.js, HTML5, Javascript, NoSQL Couch y Mongo
